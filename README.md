# NotificadorQR

Landing page para NotificadorQR — servicio de notificaciones instantáneas de pagos QR Bancolombia por WhatsApp para negocios en Colombia.

## Qué es

NotificadorQR notifica a los empleados de un negocio por WhatsApp cada vez que llega un pago. Compatible con:

- Pagos QR Bancolombia
- Transferencias por llaves Bre-B
- Transferencias cuenta a cuenta Bancolombia

## Stack

- HTML estático
- CSS (mobile-first, Inter font)
- JavaScript mínimo (smooth scroll, fade-in animations)

## Desarrollo

```bash
npx serve -l 3090
```

Abrir http://localhost:3090
